Bruce's Golf Rewards Websites

Project Info:

For this project I moved over three websites to Wordpress. Client wanted a unifed look between branch locations to foster brand recognition. I aimed to improve User Interface design by standardizing headlines, compiling information, and implementing a clear menu design. Improved User Experience by optimizing content for both mobile and desktop, and ensured information is clear and easily accessable.
